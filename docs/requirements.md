# 项目功能需求文档

## 1. 项目目标

开发一款跨平台桌面屏保应用，支持 macOS 与 Windows。用户输入出生年月日后，系统实时计算并展示退休时间，提供有趣、动态、美观的界面交互体验。

## 2. 核心功能

1. 出生日期与时间输入
   - 用户可通过日期时间选择器输入出生年月日时分。
   - 支持清空与重新选择。

2. 退休日期计算
   - 默认规则：60 周岁退休。
   - 输入变化后立即重新计算。

3. 退休时间与倒计时
   - 显示退休日期时间（满 60 周岁的同一时刻）。
   - 显示距离退休的倒计时：年、月、日、时、分、秒。
   - 倒计时数字变化时具备翻面/翻牌动画效果。
   - 显示是否已到或已过退休时间。

4. 趣味化展示
   - 动态动画（背景光球漂浮、卡片入场、倒计时卡片强调效果）。

## 3. 非功能需求

1. 跨平台兼容
   - 支持 macOS 与 Windows 的开发与打包。

2. 性能与体验
   - 前端计算实时，页面不卡顿。
   - 动画流畅，不影响主要交互。

3. 可维护性
   - 代码统一风格（ESLint + Prettier）。
   - 提交前自动检查并修复格式问题。

## 4. 后续迭代建议

- 支持按性别/工种配置退休年龄。
- 增加退休政策规则引擎（参数化配置）。
- 增加本地通知与退休提醒。
- 增加主题切换和全屏屏保模式。
